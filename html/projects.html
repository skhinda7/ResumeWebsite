Weather Prediction

Linear and polynomial regression are implemented to predict weather data for the next 10 days. As an additional method of prediction, the average is taken of both regressions for each day. The models are trained, then used to predict from which the user can save the data and display a graph for visualization. The Apache Commons Maths Library was used for polynimal curve fitting.

Maze Game

This game has been designed as a semester assignment for CS 225 (Computer Science II). The user is asked for the size of the maze, then the maze is populated with different types of tiles that have different attributes and actions, after the game is started, the agent randomly moves through the maze and the game informs the user if the agent made it to the finishing tile.
